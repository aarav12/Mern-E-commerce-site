# Mern-E-commerce-site

Ecommerce-site front end is made using React.js ,Javascript,Bootstrap,Html and css.
ecommerce-site backend is created using Node.js.
Server is created using express.js and json-web tokens are used for authentication.
Database used is MONGO-DB.

It has two different dashboard-one for user anone for the admin which are protectd by admin routes.
This site contains all the features like Sort by arrival,Most rated products,Gives suggestions for similar products,
Create and delete product(for admin) and lots more
